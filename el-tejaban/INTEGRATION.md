# El Tejaban module

POS for Cahuamanta "El Tejaban", integrated in TI Water at `/el-tejaban/`.

Source: `el-tejaban/src/` — imported via `@tejaban/*` alias.  
Mount point: `src/tejaban/tejaban-module.tsx` in the main router.

No separate `package.json` or build; use `npm run dev` / `npm run build` from `TI_water/`.

API: `TI_water_api` → `/api/v1.0/tejaban/*`
